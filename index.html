Protocol Overview

Three layers.
One cooperative web.

Tollway gives both sides what they need. Publishers get identity verification, policy control, and revenue. Agents get reliable access, structured data, and a reputational track record.

01 — POLICY

tollway.json

Sites publish a machine-readable policy file at /.well-known/tollway.json. Declare what's allowed, what costs money, and whether training is permitted. Adopt in 5 minutes.

02 — IDENTITY

Agent Headers

Every compliant agent attaches a DID (decentralized identifier), stated purpose, action scope, and an Ed25519 cryptographic signature to every request. Sites can verify who's knocking and why.

03 — ECONOMICS

HTTP 402 + x402

When the free tier runs out, sites return a price. Agents pay in USDC on Base, retry with proof. Optimistic serving keeps latency low. On-chain verification runs asynchronously. 

// Drop-in fetch replacement — handles everything automatically
import { fetch } from '@tollway/client';

const result = await fetch('https://techcrunch.com/2026/03/09/ai-news/', {
  tollway: {
    did: process.env.AGENT_DID,
    privateKey: process.env.AGENT_PRIVATE_KEY,
    wallet: process.env.AGENT_WALLET,
    purpose: 'competitive-research',
    scope: 'read',
  }
});

console.log(result.data);         // Structured JSON extraction
console.log(result.attribution);  // "TechCrunch (https://...)"
console.log(result.paid);         // true/false
console.log(result.cost);         // "0.001" USDC
// Express middleware — serve tollway.json + validate agents
import { tollwayMiddleware } from '@tollway/server';

app.use(tollwayMiddleware({
  policy: {
    freeRequestsPerDay: 100,
    pricing: [
      { action: 'read',      price: '0.001' },
      { action: 'summarize', price: '0.005' },
      { action: 'train',     price: '0.05'  },
    ],
    trainingAllowed: false,
    attributionRequired: true,
  },
  paymentAddress: process.env.WALLET_ADDRESS,
}));
// /.well-known/tollway.json
{
  "version": "0.1",

  },

Open Source Packages

Everything you need
to ship today.

npm

@tollway/client

Drop-in fetch wrapper for agent developers. Attaches identity headers, reads tollway.json, retries Tollway 402 flows, and extracts basic structured metadata.


npm i @tollway/client
npm

@tollway/server

Express and Next.js middleware for site owners. Serves tollway.json, parses agent headers, enforces freshness checks, and returns Tollway 402 payment requests.


npm i @tollway/server
community

Schema Library

CSS selector schemas for 10+ popular sites. Enables structured JSON extraction without LLM calls. Community-maintained on GitHub. PRs welcome.


github.com/tollway-protocol/tollway/schemas
